1. Создать дирекотрию. Инициализировать там гит. Прописать команду git clone.
2. Инициализировать гит в среде разработки. git add . ---> git push.
3. git pull
4.1. git status - информация о текущих настройках гит и пользователе
4.2. git log - история версий и изменений в текущем проекте
4.3. git remote -v - добавить удалённый репозиторий
